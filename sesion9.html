Diseñando la Experiencia

De la Lógica Abstracta a la Interfaz Vibrante con UI/UX

El Alma (UX) vs. El Cuerpo (UI)

Aunque viajan juntos, son dos universos diferentes. La UX es la ingeniería invisible de la intuición. La UI es el arte visible de la atracción. Una app exitosa necesita dominar ambos.

Experiencia de Usuario (UX)

Es el **PORQUÉ**. La estructura, el flujo, la lógica. Se siente, no se ve. Define si una app es útil y fácil de usar.

Interfaz de Usuario (UI)

Es el **QUÉ**. Los colores, las fuentes, los botones. Es lo que ves y tocas. Define si una app es atractiva y visualmente placentera.

Simulador Interactivo de Figma

Figma es nuestro lienzo digital. No te preocupes por sus cientos de opciones, para crear wireframes potentes solo necesitas dominar estas 4 herramientas. ¡Pasa el cursor sobre las áreas resaltadas para explorarlas!

Herramientas de Forma (R, O): ¡Tus mejores amigos! Usa el Rectángulo (R) para botones, imágenes y contenedores. Usa la Elipse (O) para avatares.
Herramienta de Texto (T): Para añadir todos los títulos, etiquetas y párrafos. Juega con el tamaño para crear jerarquía visual.
Panel de Capas: Aquí verás todos los elementos que agregas, como una lista. Es útil para organizar tu diseño y seleccionar elementos que están detrás de otros.
El Frame (F): ¡Tu lienzo! Es el primer paso. Presiona 'F' y elige un tamaño de dispositivo. Aquí es donde ocurre toda la magia del diseño.
Panel de Propiedades: Cuando seleccionas un elemento, aquí cambias sus detalles: color (Fill), borde (Stroke), alineación, etc. Para wireframes, ¡mantente en escala de grises!
